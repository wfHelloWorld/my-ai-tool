# 密钥使用说明

本应用目前直接调用阿里云百炼和 DeepSeek 提供的 AI 模型，因此在使用前需要准备并正确配置相应的 API 密钥。下面是简要说明。

## 1. 阿里云百炼密钥

阿里云为百炼平台提供了相对宽松的免费试用额度，如果只使用文字相关能力，一般可以支撑很长时间。

### 1.1 获取阿里云账号

- 使用支付宝或手机号码登录 [阿里云官网](https://www.aliyun.com/)
![阿里云登录示例](../common/img/key-doc/iShot_2026-02-16_15.25.30.png)

### 1.2 找到阿里云百炼与密钥管理

- 在阿里云控制台中搜索 [阿里云百炼](https://bailian.console.aliyun.com/cn-beijing/?spm=5176.12818093_47.overview_fav.1.125f16d0MwUAT9&tab=home#/home)
- 进入百炼控制台后，在「模型服务」区域的左下角找到「密钥管理」  
- 创建属于自己的密钥（通常是一对 AccessKey 或类似的凭证）
![密钥](../common/img/key-doc/iShot_2026-02-16_17.11.01.png)
示例：阿里云登录界面大致如下（界面风格可能会有调整，仅供参考）：


### 1.3 在本应用中配置阿里云密钥

- 打开本应用的「设置」页面  
- 找到与阿里相关的配置项（如 DASHSCOPE_API_KEY）  
- 将在百炼控制台生成的密钥完整复制粘贴到对应输入框中  
- 保存配置后即可在聊天页面或相关功能中发起请求

> 提示：阿里云百炼通常会提供大约三个月的试用期，包含一定金额（例如 50 元左右）的免费额度。如果只进行纯文本对话或简单调用，这部分额度在一年内往往都难以用完。

## 2. DeepSeek 密钥

DeepSeek 目前没有免费额度，使用前需要自行充值，费用会按实际调用量扣除。

### 2.1 注册 DeepSeek 开放平台

- 打开 DeepSeek 开放平台 [https://platform.deepseek.com/usage](https://platform.deepseek.com/usage)  
- 注册并登录自己的账号  

### 2.2 生成 API Key

- 在控制台的密钥管理页面创建新的 API Key  
- 将生成的 Key 复制下来并妥善保存

### 2.3 在本应用中配置 DeepSeek 密钥

- 在本应用的「设置」页面中找到 DeepSeek 相关配置项  
- 将刚才生成的 API Key 粘贴进去  
- 保存配置后即可在支持 DeepSeek 的模型中发起请求

> 提示：DeepSeek 需要先充值才能调用。如果应用在使用过程中报错，可以优先检查：
> - 密钥是否填错、复制不完整
> - 账号是否还有余额、是否超出限额

## 3. 常见异常与排查思路

在使用过程中，如果应用提示「调用异常」或响应失败，可以尝试：

- 打开开发者工具（Console 控制台）查看错误信息  
  - 常见错误类型：密钥无效、余额不足、权限不足、接口地址错误等  
- 检查设置页中对应的密钥是否有多余空格、换行，必要时重新复制粘贴  
- 确认所选模型与当前密钥账户权限匹配（有些模型需要单独开通）

## 4. 密钥安全建议

- 不要在公开仓库、截图或视频中展示完整密钥  
- 不要将密钥硬编码在打包后的应用中  
- 如果怀疑密钥泄露，应立即在对应平台的控制台中删除该密钥并重新生成
