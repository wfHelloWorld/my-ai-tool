## 核心功能

### 1. 图像编辑 (Editing)
支持纯文本生成，或基于上传的原图（1-4 张）和文本描述进行画面修改、风格迁移。
- **应用场景**：文生图、风格迁移、角色保持、创意合成、画面修改。
- **操作方式**：在输入框中输入描述，并可选上传 1-4 张参考图。

### 2. 智能指令扩写
内置 Prompt 智能改写功能，能够自动优化您输入的简短提示词，补充细节描述，从而生成更惊艳的画面。

## 参数说明

| 参数名称 | 说明 |
| :--- | :--- |
| **生成模式** | **图文混排 (Interleave)**：同时使用文字和图片作为输入。<br>**图像编辑 (Editing)**：基于原图进行修改（需上传参考图）。 |
| **生成数量 (n)** | 单次生成的图片数量，支持 1-4 张。 |
| **随机种子 (Seed)** | 控制生成的随机性。固定种子可复现之前的生成结果；留空则随机生成。 |
| **水印** | 是否在生成结果中添加 AI 标识水印。 |
| **指令扩写** | 是否启用智能提示词优化。建议开启以获得更好效果。 |
| **分辨率** | 支持 1280*1280, 1024*1024, 800*1200, 1200*800 等多种比例。 |

## 使用指南

1. **上传图片**（必需）：点击上传按钮添加参考图片，支持拖拽上传。必须至少上传一张图片。
2. **输入描述**：在下方对话框中输入详细的画面描述或编辑指令。支持中英文。
3. **调整参数**：按需调整生成数量、随机种子等高级选项。
4. **开始生成**：发送消息即可创建任务。生成过程可能需要几十秒，请耐心等待。

## 提示词技巧

*   **具体描述**：包含主体、风格、光影、构图等细节。例如：“一只赛博朋克风格的猫，霓虹灯光效，超高清细节，电影感构图”。
*   **参考图权重**：参考图是编辑的基础，生成的图像将保留参考图的构图和主要特征。
